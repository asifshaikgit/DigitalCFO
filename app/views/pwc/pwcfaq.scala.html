Data Security

PwC Tabulate is hosted on one of the world's most secured cloud environment of Microsoft. Some of the top banks, insurance companies and other reputed organizations use this infrastructure.

There are multiple ways by which PwC Tabulate takes care of its customer's security.

  • First and foremost, we make sure that each Login password is being encrypted so that it cannot be misused by an unauthorized user.
    The username and password which is entered by the user cannot be read IN back end.
  • While logging in, the user has to pass an image capcha test. This is done in order to prevent hacking.
  • The port which is used for data transfer between server and the browser is isolated and hence protected

If you are a user of PwC Tabulate and decide to cancel your subscription, you are required to download your data from PwC Tabulate through Microsoft Excel within 7 days of cancellation of your subscription.

After 7 days, your data will be completely and permanently deleted from the application database.
